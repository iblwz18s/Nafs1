# اختبارات محاكية نافس 2026

منصة اختبارات محاكية لاختبارات نافس الوطنية لطلاب المرحلة الابتدائية.

## المتطلبات

- Node.js & npm

## التثبيت

```sh
# استنساخ المشروع
git clone <YOUR_GIT_URL>

# الدخول لمجلد المشروع
cd <YOUR_PROJECT_NAME>

# تثبيت الاعتماديات
npm install

# تشغيل خادم التطوير
npm run dev
```

## التقنيات المستخدمة

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
